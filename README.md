## `$ ./run experiments/ --featured`

Each project below is an experiment log: hypothesis → hard part → result.

### 01 — Speech Emotion Analysis for Workplace Wellness
> **Hypothesis:** emotion can be classified from *real* workplace audio, not studio recordings.
> **Hard part:** noise. Benchmark datasets are clean; workplaces aren't. Feature selection became the whole game — MFCC, pitch, energy, and spectral features via librosa, fused into one vector per sample.
> **Result:** best classifier selected through stratified cross-validation against genuinely noisy audio.
> `Python · librosa · scikit-learn · signal processing` — [repo →](https://github.com/muhammadsudaiskhalid/Speech-Emotion-Analysis-for-Workplace-Wellness)

### 02 — Book Recommendation System
> **Hypothesis:** raw metadata is a weak feature space; engineered representations retrieve better.
> **Hard part:** combining text-based and categorical features into vectors where cosine similarity means something.
> **Result:** fast KNN retrieval over a large vectorised catalog with measurably better recommendations.
> `Python · KNN · cosine similarity · NLP · Pandas` — [repo →](https://github.com/muhammadsudaiskhalid/book-recommendation-system)

### 03 — Car Price Prediction System
> **Hypothesis:** on tabular data, clean features beat clever models.
> **Hard part:** the dataset was a mess — outliers, inconsistencies, misleading columns. EDA before modelling, every time.
> **Result:** feature selection + outlier handling + normalisation moved the metrics more than any model swap.
> `Python · regression · EDA · Pandas · NumPy` — [repo →](https://github.com/muhammadsudaiskhalid/Car-Price-Prediction-System)

### 04 — Stock Price Predictor
> **Hypothesis:** interpretable regression + good temporal features can hold their own in forecasting.
> **Hard part:** giving a stateless model a memory — lag features and rolling statistics as temporal context.
> **Result:** tuned via cross-validation, scored honestly on held-out series with MAE, RMSE, and R².
> `Python · time series · regression · Pandas` — [profile →](https://github.com/sudais-khalid)

### 05 — AI Storyteller Genesis
> **Hypothesis:** prompt-driven architecture can keep long generative narratives coherent and genre-aware.
> **Hard part:** thematic consistency across open-ended generation — structure in the prompt, not luck in the sampling.
> **Result:** controllable, contextually consistent story generation.
> `Python · NLP · generative AI` — [profile →](https://github.com/sudais-khalid)
